# Vue Course System Prototype

This directory is a standalone Vue 3 + Vite prototype for course management. It is not imported or routed by the current React/Umi admin frontend.

## Status

- Independent project: yes.
- Used by current admin frontend: no.
- Recommended handling for GitHub release: keep only if you want to show the historical prototype; otherwise move it to a separate repository or delete it after confirmation.

## Startup

```bash
cd smartclass-manage-frontend-main/vue-course-system
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

The current production admin application is `smartclass-manage-frontend-main` itself, not this prototype.
